# DSA Practice: Striver's A2Z Roadmap

This repository contains multi-language implementations of DSA curriculum. Each problem is solved in **C++**, **Java**, and **Python** to ensure cross-language proficiency.

## 📂 Repository Structure
- **Cpp/**: Source code for C++ implementations.
- **Java/**: Source code for Java implementations.
- **Python/**: Source code for Python implementations.
- **Difficulty:** Categorized from Beginner to Expert levels.

---

## 🗺️ Roadmap: Topic Overview

| Step | Topic | Sub-topics Included |
| :--- | :--- | :--- |
| 1 | **Learn the Basics** | I/O, STL/Collections, Basic Maths, Recursion, Hashing |
| 2 | **Sorting Techniques** | Selection, Bubble, Insertion, Merge, Quick, Recursive Sorting |
| 3 | **Arrays** | Easy, Medium, and Hard array manipulation |
| 4 | **Binary Search** | 1D, 2D Arrays, and Search Space algorithms |
| 5 | **Strings** | Basic and Medium level String processing |
| 6 | **LinkedList** | Singly, Doubly, Circular, and complex LL problems |
| 7 | **Recursion** | Subsequences, Combination Sum, and Pattern-wise logic |
| 8 | **Bit Manipulation** | Bitwise operators, Set/Unset bits, Power sets |
| 9 | **Stacks & Queues** | Implementations and Monotonic Stack/Queue patterns |
| 10 | **Sliding Window & Two Pointers** | Fixed and Variable Window, Longest Substring |
| 11 | **Heaps** | Priority Queues, K-th element problems |
| 12 | **Greedy Algorithms** | Optimization problems and Activity Selection |
| 13 | **Binary Trees** | Traversals (Pre/In/Post), View problems, Hard Tree logic |
| 14 | **Binary Search Trees** | BST properties, Construction, and Validation |
| 15 | **Graphs** | BFS, DFS, Shortest Paths (Dijkstra/Bellman), MST |
| 16 | **Dynamic Programming** | Subsets, Strings, Grids, LIS, MCM, and Partition DP |
| 17 | **Tries** | Prefix Trees and XOR problems |
| 18 | **Strings (Hard)** | Advanced Pattern Matching and String DP |
